  7. Commandement 7: Nourrissez les baleines de votre amour, mais laissez les containers se nourrir de marchandises.

  11. Commandement 11: Les baleines peuvent chanter, les containers peuvent contenir, chacun son talent !
  12. Commandement 12: Les containers n'aiment pas être pris pour des baleines, acceptez-les comme ils sont.
  13. Commandement 13: La baleine bleue est le plus grand animal du monde, le container bleu est le plus grand du port.
  14. Commandement 14: Pour devenir ami avec une baleine, c'est compliqué, avec un container, c'est facile !
  15. Commandement 15: Les containers sont excellents pour le rangement, mais ne les utilisez pas pour stocker les poissons.
